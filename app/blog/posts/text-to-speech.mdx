It Uses Browser Inbuilt Web Speech API To Convert Text Into The Speech.

Text to Speech webapp uses Browser inbuilt Web Speech API to convert
Text into the Speech.

Service provides different features to control Pitch and Rate of Voice

Some Features -
- Enter your Text and click on Play Button or
- To listen the specific line of the Paragraph Select the line and
  click on play option.
- Browser provides different speech models user can choose one of
  Then if user doesnot choose any Voice Default voice is Microsoft
  David Desktop - English (United States)
- Voice modal availability is entirely depend on the Browser.
- Different Browsers support Different language as per the region or
  availability
