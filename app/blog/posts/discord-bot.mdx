A Python programing language Interpreter bot for Discord. Beginners can
easily host and integrate discord with bot

## Discord Bot That Run Python Programs

Discord Bot that capable to run python programs right from your chat in
discord server specific channel .

You have to only connect the bot from the server and set the channel
name

For better understanding read each file of repository befor using bot

### Run Locally

Create directory
```bash
mkdir DiscordBot
cd DiscordBot
```

Clone the project
```bash
git clone https://github.com/innovatorved/discord-bot-that-run-Python-Program.
git .
```

Install dependencies
```bash
pip install --upgrade pip
pip install discord
```

Start the Bot
```bash
python discord-bot-that-run-python-programs.py __channelName__ __botToken__
```
