## What Makes It Special?

### üîí Screen Share Protection

The killer feature: **the app window is completely hidden during screen sharing**. Share your screen on Zoom, Meet, or Teams, and it stays invisible. No awkward explanations, no scrambling‚Äîit just disappears from their view while staying functional on yours.

## Getting Started

### Audio Setup

You need audio loopback to capture system audio:

- **macOS**: [BlackHole](https://existential.audio/blackhole/) + aggregate device
- **Windows**: [VB-Audio Cable](https://vb-audio.com/Cable/) or [Voicemeeter](https://vb-audio.com/Voicemeeter/banana.htm)
- **Linux**: PulseAudio/PipeWire with `helvum` or `qpwgraph`

Check the [README](https://github.com/innovatorved/realtime-interview-copilot/tree/nightly) for detailed setup instructions.

### Installation

```bash
git clone -b desktop-app https://github.com/innovatorved/realtime-interview-copilot.git
cd realtime-interview-copilot
pnpm install
pnpm electron:dev  # or pnpm electron:build for production
```

### Usage

1. Launch app ‚Üí Configure audio device
2. Enter interview context (role, company, background)
3. Click mic ‚Üí Start transcription
4. Choose Copilot or Summarizer mode
5. Hit Enter ‚Üí Get AI responses
6. Press `K` ‚Üí Ask follow-up questions anytime
